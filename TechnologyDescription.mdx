# Technology Description

1. We decided to develop our project using C #. Although not all of us have extensive knowledge of the language, we felt it was similar to Java and widely used in the current market.
2. For a framework, we decided to go with Blazor Server. We based our decision of the framework through our research of user reviews, Blazor was seen by most users as Intuitive with its combination of client-side Interactivity and server-side rendering in a single programming model.

   ```plaintext
   https://dotnet.microsoft.com/en-us/apps/aspnet/web-apps/blazor
   ```
3. Our storage plan is to utilize SQLite, which we decided on due to members of our team having pre-existing experience.
4. Since all members own a Windows OS Laptop, our operating environment will be in Windows
